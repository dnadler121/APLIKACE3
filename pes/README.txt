BASKERVILLE FLASK ČTENÁŘSKÁ HRA

Spuštění:
1) pip install -r requirements.txt
2) python app.py
3) otevři http://127.0.0.1:5000

Obsah:
- 15 částí příběhu.
- Po správné odpovědi se odemkne další část.
- 3 životy.
- Tlačítko Číst nahlas používá hlas v prohlížeči.
- Funguje na PC, Androidu i iPhonu přes prohlížeč.

Poznámka:
Na mobilech může být nutné kliknout na tlačítko Číst nahlas, protože prohlížeče blokují automatické přehrávání hlasu.
