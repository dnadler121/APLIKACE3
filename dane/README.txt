
VÝPLATA HPP / DPP / DPČ – výuková Flask aplikace

Spuštění:
1) Rozbal ZIP.
2) Otevři terminál ve složce projektu.
3) Nainstaluj Flask:

   pip install -r requirements.txt

4) Spusť aplikaci:

   python app.py

5) V prohlížeči otevři:

   http://127.0.0.1:5000

Poznámka:
Aplikace používá zjednodušený školní model výpočtu mzdy pro výuku.
Nejde o plnohodnotnou mzdovou kalkulačku.
