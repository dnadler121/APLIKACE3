# Nerovnice s neznámou ve jmenovateli – Flask aplikace

## Spuštění

```bash
pip install flask
python app.py
```

Otevři:

```text
http://127.0.0.1:5000
```

## Co aplikace umí

- generuje 10 příkladů typu `(x-a)/(x-b) > 0` nebo `< 0`,
- znaménko se pravidelně střídá,
- student vyplní celý rozpad na dva průniky,
- student upraví nerovnice na tvar `x>...` nebo `x<...`,
- student zakreslí oba průniky na číselné ose,
- student zapíše výsledek,
- dva intervaly ve výsledku lze oddělit čárkou.
